## 1. Project Overview

I Build this website to talk about marine life and its importance. It includes five pages with information about ecosystems, threats, and how people can help. It uses neutral and bright colours to represent the oceans of our world, it also contains images, videos, a table, and a form along with the site informations.

##2. How do the pages work together?

The pages have a simple and continous structure, with a introduction, some deep in on the theme ("Why oceans matter", "marine life" and "Threats") and a way to the user interact with the website in the last page.

## 3. Technical Implementation

### HTML

Key techniques used:

- Semantic structure: header, nav, main, section, footer on every page  
- Tables: Used on page2 to organize animal data  
- Forms: Contact form on page5 with inputs and textarea  
- Lists: Ordered and unordered lists for information  
- Images: All images include alt text for accessibility  

### Code structure

Each page follows the same structure:
- Header with title  
- Navigation bar  
- Main content divided into sections  
- Footer  


### CSS

### Organization

- Reset and root variables (colors)  
- General styles (body, text)  
- Layout (flexbox, grid)  
- Components (buttons, forms, tables)  
- Responsive design (media queries)  


### Key techniques

- Flexbox for layout (navigation and content sections)  
- Grid for structured sections (page5)  
- Hover effects on links and buttons  
- Box shadows and border radius for design  
- Media queries for responsiveness  


### CSS Flexbox

Used for:
- Navigation bar (horizontal layout)  
- Image + text sections  
- Facts section layout  

Why: Flexbox is good for aligning elements in a row and controlling spacing.


### CSS Grid

Used for:
- Page5 grid layout (two sections side by side)

Why: Grid is useful for dividing content into columns.


### Responsive Design

- Desktop: Horizontal layout, images side by side with text

## 4. Design Choices

### Color Scheme

- Primary: #0077b6 – ocean blue  
- Secondary: #023e8a – darker ocean tone  
- Accent: #00b4d8 – highlights and interactions  

### Typography

- Headings: Segoe UI bold  
- Body: Segoe UI  

### Why it works

The blue color palette represents the ocean and the creates of the sea, a calm and consistent colour as blue and whtie it's just perferct for the theme. Feels very appropriate to use these type of colours


### Design Principles

**Contrast:**  
Headings are larger and darker, buttons use accent colors  

**Repetition:**  
Same layout, colors, and navigation across all pages  

**Alignment:**  
Content is centered in a container with structured spacing  

**Proximity:**  
Related content is grouped into sections
